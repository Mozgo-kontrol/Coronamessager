# Coronamessager

Hi! I'm Igor, This is a uni projekt. We have created it with **Java** for backend and **Firebase** for storing data.
<br>

## Features

- Register or login as an user.
- Update user account.
- Apply filters for searching distance.
- Find user lokation
- save user info to Firebase


##  NPM Packages Used

- Java
- Material UI
- Firebase
- XML
- Gradle
